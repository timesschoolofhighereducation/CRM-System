\section{1) Executive Summary}

This document provides a full commercial and technical cost estimate for delivering and operating the Education CRM web application in Sri Lanka. The estimate includes:\\

\begin{itemize}
\item Discovery and planning
\item UI/UX and frontend implementation
\item Backend and API implementation
\item Database and infrastructure setup
\item Security and role/permission management
\item Reporting and export modules (Excel/PDF)
\item Notification and communication modules
\item Testing and quality assurance
\item Deployment, handover, and training
\item Ongoing maintenance and support
\end{itemize}

\subsubsection{Total Project Investment (One-Time)}

\begin{itemize}
\item \\textbf{Lean delivery model:} \\texttt{LKR 3,450,000 - 4,950,000}

\section{10) Risk Factors Affecting Cost}

Factors that can increase budget:\\

\begin{itemize}
\item Frequent scope changes during development
\item New third-party integrations not in initial scope
\item Large legacy data migration requirements
\item High compliance/security audit requirements
\item Tight deadline with parallel team scaling
\item Multi-language or multi-tenant refactoring
\end{itemize}

Factors that can reduce budget:\\

\begin{itemize}
\item Strict MVP scope control
\item Reuse of existing stable modules
\item Strong and fast client UAT participation
\item Fewer environments and integrations
\end{itemize}
