# User Handbook
## Education CRM Application

This handbook explains how to use the main features of the Education CRM in simple steps.

## 1) Getting Started

### Sign In
1. Open the application login page.
2. Enter your email and password.
3. Click **Sign In**.

### Important Note on Access
- The system uses role-based permissions.
- You only see pages and actions allowed for your role.

## 2) Dashboard Overview

After login, the dashboard helps you quickly view:
- key numbers (KPI cards)
- pending follow-ups
- task status
- recent activity

Use this page as your daily starting point.

## 3) How to Manage Student Inquiries

### View and Search Inquiries
1. Open **Inquiries**.
2. Use search and filters to find records.
3. Use pagination to move through pages.

### Create a New Inquiry
1. Click **Add Inquiry** (or similar button).
2. Fill required student details.
3. Save the record.

### Update Inquiry Status
1. Open the inquiry details page.
2. Change status (for example: new, in progress, completed).
3. Add notes about student interactions.
4. Save updates.

## 4) How to Manage Tasks and Follow-Ups

### Task Dashboard
1. Open **Tasks**.
2. Review tasks assigned to you.
3. Check due dates and priorities.

### Kanban Workflow
1. Move tasks between columns such as:
   - To Do
   - In Progress
   - Done
2. Keep tasks updated daily.

### Follow-Up Reminders
- Check “due today” and upcoming items.
- Complete follow-ups on time and mark them done.

## 5) How to Use Calendar and Meetings

### Calendar
1. Open **Calendar**.
2. Select a day to see all events.
3. Review meetings, follow-ups, and task-related dates.

### Meetings
1. Create a new meeting.
2. Add date/time, location or online link, and notes.
3. Save and monitor upcoming meetings.

## 6) How to Run Campaigns

### A) Standard Campaigns
1. Open **Campaigns**.
2. Create or edit a campaign.
3. Track campaign status and performance fields.

### B) Promotion Codes
1. Open **Promotion Codes**.
2. Create or edit code details.
3. Track promoter and code performance.

### C) WhatsApp Campaigns
1. Open **WhatsApp Campaigns**.
2. Select recipients using filters (program, date, search, etc.).
3. Write the message and add media if needed.
4. Use templates if available.
5. Send campaign and review sent/failed results.

### D) Email Campaigns
1. Open **Email Campaigns**.
2. Select recipients with valid email addresses.
3. Add subject, message, and attachments if needed.
4. Send and review delivery results.

## 7) Social Media Post Approval Flow

1. Open **Social Media Posts** (or approval module).
2. Create a post with caption, media, and schedule.
3. Submit for approval.
4. Approver actions:
   - Approve
   - Reject with comment
5. If rejected, update the post and resubmit.

## 8) Reports and Analytics

### Dashboard and Reports
1. Open **Reports/Analytics**.
2. Apply filters (date range, user, campaign).
3. Review KPI cards, charts, and activity logs.
4. Export data if needed.

### Weekly and Annual Reports
- Use weekly reports for regular team tracking.
- Use annual reports for higher-level review (admin-focused).

## 9) Data and Content Management

Depending on your role, you can manage:
- Programs and Levels
- Program descriptions
- Q&A / FAQ entries
- Projects and Deals
- Notebooks and Notes

Always save changes and verify entries after updates.

## 10) Users, Roles, and Settings (Admin/Authorized Users)

### Users and Roles
1. Open **Users** or **Roles & Permissions**.
2. Add/edit users.
3. Assign the correct role and permissions.

### Settings
- Update system/account settings based on access rights.
- Avoid changing critical settings without approval.

## 11) Notifications and Reminders

- Review in-app notifications regularly.
- Use unread indicators to track pending items.
- Mark single notifications or all as read.
- Use deep links in notifications to open related pages quickly.

## 12) AI Assistant and Productivity Tools

### AI Assistant
1. Open **AI Assistant**.
2. Enter your question or content request.
3. Review and use the generated response in your workflow.

### Mini Games (Optional)
- Use mini games for short mental breaks.
- This does not affect business data.

## 13) Trash Bin and Recovery

- Open **Trash Bin** (if your role allows it).
- Restore deleted inquiries/campaigns when needed.
- Review restored data for accuracy.

## 14) Daily Best Practices

- Start from dashboard every morning.
- Check follow-ups due today first.
- Keep inquiry status and notes updated immediately.
- Update task progress before end of day.
- Review notifications and pending approvals.
- Use filters in reports for weekly performance checks.

## 15) Troubleshooting Tips

- If a page or button is missing, check your role permissions.
- If campaign sending fails, verify recipient data and message format.
- If login issues occur, confirm credentials or contact admin.
- If data looks incorrect, refresh filters and recheck date range.

## 16) Support and Escalation

For technical support, contact your system administrator or internal support team with:
- module name
- short issue description
- screenshot (if available)
- time of issue

This helps resolve problems faster.
