## 1. Executive Summary

This report presents a professional, client-ready cost estimate for developing, deploying, and operating the Education CRM web application in Sri Lanka.

The estimate covers:

- Discovery and requirement finalization
- UI/UX design and frontend implementation
- Backend APIs and data services
- Database design and optimization
- Security and role-based authorization
- Dashboard, reports, and Excel/PDF exports
- QA, UAT, deployment, and handover
- Post-launch maintenance and support

### One-Time Implementation Cost (CAPEX)

- **Lean model:** `LKR 3,450,000 - 4,950,000`
- **Standard model (recommended):** `LKR 5,250,000 - 7,950,000`
- **Enterprise model:** `LKR 8,250,000 - 12,500,000`

### Monthly Operational Cost (OPEX)

- **Basic support and hosting:** `LKR 95,000 - 180,000 / month`
- **Managed support (recommended):** `LKR 180,000 - 375,000 / month`
- **Enterprise SLA support:** `LKR 375,000 - 900,000+ / month`

## 3. Assumptions

1. The current codebase is reusable and can be stabilized.
2. The delivery team is based in Sri Lanka.
3. The stack remains Next.js + TypeScript + Prisma + SQL database.
4. Client-side UAT participation is available on schedule.
5. Third-party licenses and API usage are billed separately.
6. Taxes and statutory fees are excluded unless explicitly included in the final commercial quote.

## 10. Risk Factors That Affect Cost

### Cost Increase Risks

- Frequent scope changes
- New integrations introduced mid-project
- Significant data migration complexity
- Elevated compliance/security requirements
- Aggressive deadlines requiring parallel teams
- Multi-language or multi-tenant redesign

### Cost Reduction Enablers

- Strict MVP scope control
- Reuse of stable existing modules
- Fast client UAT turnaround
- Limited integrations and environment complexity
