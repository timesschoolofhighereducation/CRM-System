### 4️⃣ Multiple Programs = Multiple Inquiries 🎉

**THE BIG CHANGE:**

#### Before:
```
Visitor: John Doe
Programs: BSc Business, BSc IT, BSc Supply Chain
         ↓
Click "Create Inquiry"
         ↓
Creates 1 inquiry with 3 programs
         ↓
❌ Problem: Who handles this? All 3 coordinators see same inquiry!
```

#### Now:
```
Visitor: John Doe  
Programs: BSc Business, BSc IT, BSc Supply Chain [3 programs]
         ↓
Click "Create 3 Inquiries"
         ↓
Creates 3 separate inquiries:
  1️⃣ Inquiry for BSc Business → Business Coordinator
  2️⃣ Inquiry for BSc IT → IT Coordinator  
  3️⃣ Inquiry for BSc Supply Chain → Supply Chain Coordinator
         ↓
✅ Each coordinator handles their own program!
```

## 🎬 Step-by-Step Usage

### Scenario: Convert a visitor with 3 programs

1. **Find the visitor**
   ```
   Option A: Type name in search bar
   Option B: Filter by program
   Option C: Filter by date range
   ```

2. **Check program count**
   ```
   Look for badge: [3 programs]
   ```

3. **Click the button**
   ```
   Button says: "Create 3 Inquiries"
   Hover to see: "Will create 3 separate inquiries (one for each program)"
   ```

4. **Fill the form**
   ```
   - Name: (pre-filled from registration)
   - Phone: (pre-filled)
   - Programs: (automatically detected)
   - Marketing Source: Exhibition Registration
   - Other details: (fill as needed)
   ```

5. **Submit**
   ```
   ✅ Success message: "Successfully created 3 inquiries (one for each program)"
   ✅ Row turns red (converted)
   ✅ Button changes to "Converted"
   ```

6. **What happens behind the scenes**
   ```
   System creates:
   
   Inquiry #1:
   - Name: John Doe
   - Program: BSc Business
   - Assigned to: Business Coordinator
   
   Inquiry #2:
   - Name: John Doe
   - Program: BSc IT
   - Assigned to: IT Coordinator
   
   Inquiry #3:
   - Name: John Doe  
   - Program: BSc Supply Chain
   - Assigned to: Supply Chain Coordinator
   ```

## 🆘 Troubleshooting

### Problem: "This visitor has already been converted"
**Solution:** The visitor was already processed. Check the "All Inquiries" tab to see the created inquiries.

### Problem: "This visitor has no programs selected"
**Solution:** Contact the visitor to select programs first, or manually add programs before converting.

### Problem: Can't find a visitor
**Solution:** 
1. Clear all filters (click "Clear" button)
2. Try searching by phone number instead of name
3. Check date range filter - might be too narrow

### Problem: Button says "Create 5 Inquiries" but I only want 1
**Solution:** This is by design! Each program coordinator needs their own inquiry. All 5 will be created automatically.
